From www to WWW.......

The web has always amazed me, esp. so because I spend most of my waking life in and around it. So, I got curious to know what goes on behind the scenes and how does it really work. I like to call it the world wizard coz it seems to know everything ;) and one of my interactions with the world wizard led me to discover a course being offered about the wonders of this world wizard, this summer at Berkeley. So I decided to take it and widen my perspective of how www (wonder of the world wizard) works and learn more about WWW (World Wide Web)

I have had one week of class so far and things look pretty darn cool !! Turns out that we will be able to build our own website by the end of the course. We even have a course website to keep track of stuff we are learning.

Behind the scenes

Looks like there is a lot going on back here and there is a lot to learn. For now the list seems pretty long :)

  • How the web works - Client Server Interaction
  • The language client server use to talk - HTTP
  • the basics - HTML
  • the next big thing - HTML5
  • Various browsers and tools - Firefox, Chrome, IE
  • and lots more !

Since there is so much going on, there are some standards that govern stuff. Two most important bodies to refer to for standard are -

  1. W3C - World Wide Web Constorium
  2. WHATWG - Web Hypertext Application Technology Working Group
